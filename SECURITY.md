# Security Policy

Pano is currently published as a Community Preview.

## Supported version

| Version | Support status |
| --- | --- |
| 1.0.52.2 Community Preview | Security reports accepted |

## Reporting a vulnerability

Please do not open a public issue for exploitable vulnerabilities. Send a private report to the repository owner using GitHub private vulnerability reporting if it is enabled, or contact the maintainer directly.

Include:

- affected Pano version,
- minimal reproduction steps,
- expected impact,
- affected platform and .NET SDK version.

The maintainer will triage reports before publishing fixes or advisories.
