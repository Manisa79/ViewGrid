# Taylan.Pano 1.0.51.8 Community Preview

## Documentation Capture Scenario Matrix

Bu sürüm, TestApp içindeki Documentation Capture Mode ekranını özellik/senaryo bazlı dokümantasyon ekran görüntüsü üretimine genişletir.

### Öne çıkanlar

- Aynı örnek formdan birden fazla özellik görüntüsü üretilebilir.
- ViewMode, Media, Audix, Playback, Filtering, Theme, Export ve Pro Experience senaryoları ayrı capture item olarak listelenir.
- Manifest, markdown ve DOCX insert map dosyaları kategori ve senaryo bilgisi içerir.
- Capture akışı, sekme/combo/buton/listbox gibi yaygın UI öğelerini metin veya isim üzerinden genel şekilde seçmeye çalışır.

### Not

Bu ortamda dotnet build çalıştırılamadığı için build testi kullanıcı makinesinde yapılmalıdır.
