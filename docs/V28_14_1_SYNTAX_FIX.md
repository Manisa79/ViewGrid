# ViewGrid v28.14.1 - Syntax Fix

Bu sürüm v28.14 platform/filtre altyapısında kalan C# sözdizimi hatasını düzeltir.

## Düzeltilenler

- Formula engine içindeki tek tırnak temizleme kodu C# uyumlu hale getirildi.
- `Geçersiz ifade terimi`, `Sözdizimi hatası`, `Sabitte yeni satır karakteri`, `no/left adı geçerli bağlamda yok` gibi zincirleme hataların sebebi giderildi.
- v28.14 Dashboard/CardView filtre kolon seçimi, Quick Filter, row height ve poster/card geçiş düzeltmeleri korundu.

## Not

Visual Studio tarafında eski obj/bin kalıntısı varsa temiz build için `bin` ve `obj` klasörlerini silin.
