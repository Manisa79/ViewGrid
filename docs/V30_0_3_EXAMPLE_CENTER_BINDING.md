# ViewGrid v30.0.3 - Visual Experience Suite Example Center Binding

Bu güncellemede v30 görsel modları sadece çekirdekte kalmayacak şekilde örnek merkezine bağlandı.

## Görünür hale getirilen yerler

- Ana uygulama üst menüsü: `Örnekler > V30 Visual Experience Suite`
- Ana uygulama üst menüsü: `Örnekler > V30 Gelişmiş Görünüm Vitrini`
- Ana uygulama üst menüsü: `Görünüm` menüsüne yeni v30 modları
- `SampleHubForm / Example Center Pro` sol menüsünde ayrı `v30 Visual Experience Suite` bölümü
- Ortak sample navigation menüsünde V30 kısa yolları

## Menüden açılan ana örnekler

- `MediaLibrarySampleForm`
  - Poster
  - Gallery
  - MediaTile
  - FilmStrip
  - KPI Dashboard
  - HeatMap
  - MiniChart
  - PropertyCard
  - DetailCard
  - Görsel ölçekleme: Cover / Contain / Stretch
  - Tip filtresi ve arama

- `ViewModeShowcaseSampleForm`
  - Details
  - Poster
  - Gallery
  - Kanban
  - Timeline
  - RowPreview
  - GroupCard
  - PropertyCard
  - KpiDashboard
  - HeatMap
  - MiniChart
  - MasterDetail

## Korunan düzeltme

Önceki pakette düzeltilen `DrawRoundedRectangle(..., int radius)` uyumsuzluğu korunmuştur.
