# Taylan.Pano v27.9 - Popular Enterprise Features

Bu ara sürüm, popüler WinForms grid bileşenlerinde sık kullanılan özellikleri Pano tarafında tek preset altında toplar.

## Eklenen ana paket

- `EnablePopularFeaturePack`
- `PopularFeaturePreset`
- `ApplyPopularFeaturePack(...)`
- `AddSemanticStatusConditionalFormat(...)`
- `AddNumericSummary(...)`
- `AddCountSummary(...)`

## Presetler

- `Balanced`
- `MasterData`
- `SupportDesk`
- `LargeDataReview`
- `DataEntry`

## Hedeflenen kullanım

### MasterData
- Yoğun liste
- İlk kolon dondurma
- Geniş filtre popup
- Column chooser
- Özet footer
- Uzun SAP/BOM metinleri için tooltip ve auto-width filter

### AOI Support Desk
- Dashboard/card görünüm
- Üst hızlı filtre barı
- Floating filtre butonu
- Aktif filtre chipleri
- Durum rengi conditional formatting

### Büyük veri inceleme
- Details/Dense görünüm
- Büyük filter scan limitleri
- Modern search panel
- Highlight search

### Veri giriş ekranları
- F2 inline edit
- Footer summary
- Full header/body context menu

## Not

Bu sürüm yeni motor eklemekten çok, daha önce eklenen yetenekleri designer/runtime tarafında tek komutla kullanılabilir hale getirir.
